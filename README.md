## Projects

### [Vega](https://github.com/MustafaNazeer/Vega)

Quantitative options pricer. Black Scholes (analytical and NumPy vectorized), Cox Ross Rubinstein binomial, and Monte Carlo with antithetic variates. Six closed form Greeks, heat map shock visualizations on HTML canvas, and a backtesting engine over yfinance daily closes. Auth0 OAuth with backend RS256 JWT verification and per user calculation history on Postgres. 505 tests on a CI pipeline running ruff, mypy, pytest, vitest, bandit, semgrep, and gitleaks.

Python, FastAPI, React, TypeScript, PostgreSQL, Auth0, Docker. [Live demo](https://vega-2rd.pages.dev/).

### [AdrenalineAI](https://github.com/MustafaNazeer/AdrenalineAI)

UFC fight prediction platform. Scrapes 4,400+ fighter records from UFCStats.com with incremental updates, builds 31 engineered features including rolling averages, win streaks, and live betting odds from The Odds API, and predicts fight outcomes with an XGBoost classifier at 81% cross validation accuracy.

Python, Streamlit, XGBoost, BeautifulSoup, Pandas. [Live demo](https://adrenalineai.streamlit.app).

### [SpotMe](https://github.com/MustafaNazeer/SpotMe)

Mobile first fitness tracking PWA. Offline workout logging via IndexedDB that auto syncs to Supabase when connectivity returns, row level security on a normalized 5 table schema, and 136 Vitest tests across calculations, Supabase query mocking, offline sync, and input validation.

TypeScript, Next.js, Supabase, Tailwind, Vitest.

### [BaselineMS](https://github.com/MustafaNazeer/BaselineMS) (currently building)

Native Android app for people living with multiple sclerosis. Weekly battery of five self administered tests with on device gait analysis from 30 seconds of phone IMU data. All data stays on device; the app does not declare the INTERNET permission.

Kotlin, Jetpack Compose, Room.
